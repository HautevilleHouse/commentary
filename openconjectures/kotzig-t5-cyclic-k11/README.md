# Kotzig 5-Edge T-Tree Cyclic Decomposition in K11

The tree has edges `(0,1), (0,2), (0,3), (3,4), (4,5)`. Base labels in `Z/11Z` are `{0:0, 1:1, 2:2, 3:3, 4:10, 5:4}`. Eleven cyclic shifts partition all 55 edges of `K11`.

This is one finite Kotzig slice only; the general conjecture remains open.
