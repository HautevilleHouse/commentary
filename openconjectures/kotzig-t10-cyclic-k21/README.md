# Kotzig 10-Edge T-Tree Cyclic Decomposition in K21

For the tree with edges `(0,1),(0,2),(0,3),(3,4),(4,5),(5,6),(6,7),(7,8),(8,9),(9,10)`, base labels in `Z/21Z` are `{0:0,1:9,2:10,3:8,4:1,5:7,6:2,7:6,8:3,9:5,10:4}`. Twenty-one cyclic shifts partition all 210 edges of `K21`.

This is one finite Kotzig slice only; the general conjecture remains open.
