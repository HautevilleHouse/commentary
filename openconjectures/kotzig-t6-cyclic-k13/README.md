# Kotzig 6-Edge T-Tree Cyclic Decomposition in K13

For the tree with edges `(0,1),(0,2),(0,3),(3,4),(4,5),(5,6)`, base labels in `Z/13Z` are `{0:2,1:3,2:4,3:5,4:1,5:6,6:0}`. Thirteen cyclic shifts partition all 78 edges of `K13`.

This is one finite Kotzig slice only; the general conjecture remains open.
