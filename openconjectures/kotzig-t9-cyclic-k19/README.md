# Kotzig 9-Edge T-Tree Cyclic Decomposition in K19

For the tree with edges `(0,1),(0,2),(0,3),(3,4),(4,5),(5,6),(6,7),(7,8),(8,9)`, base labels in `Z/19Z` are `{0:0,1:1,2:2,3:3,4:7,5:12,6:6,7:13,8:5,9:14}`. Nineteen cyclic shifts partition all 171 edges of `K19`.

This is one finite Kotzig slice only; the general conjecture remains open.
